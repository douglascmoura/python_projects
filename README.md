# 🐍 Python Projects & Data Apps

Este repositório é uma coletânea centralizada dos meus projetos, scripts e aplicações desenvolvidas em **Python**. 

O objetivo deste espaço é documentar a minha jornada de desenvolvimento na linguagem, abrangendo desde a construção de lógica de programação fundamental até a criação de painéis interativos e aplicações web voltadas para a visualização de dados.
